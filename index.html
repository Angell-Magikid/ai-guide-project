LV 1
🗺️

静态展示

只有图文，用户只能看

⏱ 1-2小时 · 免费
LV 2
📝

表单收集

用户可以填写表单

无人机版： 预约配送表单（地址 + 时间段）+ 输入订单号查询状态
⏱ 2-4小时 · 免费
LV 3
⭐ 推荐

动态交互

地图、实时追踪、动画效果

⏱ 半天-1天 · 免费

💾

真实订单系统 + 用户登录

无人机版： 顾客账号 + 实时GPS追踪 + 配送管理后台
⏱ 3-7天 · $0-50/月

✅ drone-hero.jpg

❌ IMG_20240101.jpg
❌ 图片 1.png

✅ 格式：MP4 (H.264)
✅ 大小：< 30MB
✅ 大视频 → YouTube 嵌入

发布上线

GitHub Pages 全流程

🚁 案例：让全世界都能看到无人机追踪网站
1
👤

注册 GitHub 账号

⏱ 5分钟 · 免费

访问 github.com，用你的邮箱注册。用户名会出现在网址里，建议用品牌名：walmart-drone

2
📦

创建仓库（Repository）

⏱ 3分钟

点「New repository」→ 命名：drone-delivery → 选 Public → 创建

3
📤

上传项目文件

⏱ 10分钟

在仓库页面点「uploading an existing file」→ 把项目文件夹内容全部拖入上传

⚠️ 注意：不要上传 .workbuddy 文件夹
4
🚀

开启 GitHub Pages

⏱ 2分钟

仓库 → Settings → Pages → Source 选「Deploy from a branch」→ Branch 选 main → Save

等1-3分钟后，网站就在：https://walmart-drone.github.io/drone-delivery/

可选
5
🌐

绑定自定义域名

⏱ 15分钟 + DNS等待 · $10-15/年

在 Namecheap/GoDaddy 买域名 → DNS 设置 CNAME → Pages 填域名 → 勾选 HTTPS

🎉 效果：https://dfw-drone.walmart.com

📌 发布前必须人工检查的事情

📝文案检查

联系方式对吗？服务区域描述准确吗？

📱手机测试

用真实手机打开，检查地图和按钮是否好用

🔗按钮检查

所有链接和按钮点击后能正常工作

🔒隐私声明

如果收集用户信息，需要隐私政策

📊数据统计

加入 Google Analytics 了解访问量

🖼️版权检查

无人机图片是否有使用授权
